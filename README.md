# COI
A Video Sharing Web Application  
**Tech Stack:** Golang, MySQL, Docker
## Main feature
- Supporting user/creator/admin account with RBAC.
- Video uploading.
- User interaction: comments, likes, subscribes,...
- Creating user channel, user playlists.
- Notification
